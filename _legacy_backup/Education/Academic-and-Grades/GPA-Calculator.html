1. Enter the course name (optional).

Grading Scale

This calculator uses the standard 4.0 scale:

  • B+ = 3.3
  • B = 3.0
  • B- = 2.7
  • C+ = 2.3
  • C = 2.0
  • C- = 1.7
  • D+ = 1.3
  • D = 1.0
