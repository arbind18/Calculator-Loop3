Z-Score Calculator

Calculate the Standard Score (Z-Score)

-

Formula

$$Z = \frac{x - \mu}{\sigma}$$

Where:

$x$ = Raw Score (the value you are evaluating)

$\mu$ = Population Mean

$\sigma$ = Population Standard Deviation

Interpretation:

A Z-score indicates how many standard deviations an element is from the mean.

  • Z > 0: The value is above the mean.
  • Z < 0: The value is below the mean.
  • Z = 0: The value is equal to the mean.
