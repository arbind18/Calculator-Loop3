How It Works

Addition/Subtraction: Vectors are added or subtracted component-wise.

Scalar Multiplication: Each component of the vector is multiplied by the scalar $k$.

Magnitude: The length of the vector, calculated using the Pythagorean theorem: $|\mathbf{v}| = \sqrt{x^2 + y^2 + z^2}$.

Normalization: Finding the unit vector in the same direction: $\hat{\mathbf{v}} = \frac{\mathbf{v}}{|\mathbf{v}|}$.
