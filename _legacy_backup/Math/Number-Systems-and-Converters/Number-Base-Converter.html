Common Bases

Binary (Base 2): Uses 0 and 1. Fundamental language of computers.

Octal (Base 8): Uses digits 0-7. Often used in Unix file permissions.

Decimal (Base 10): Uses digits 0-9. The standard system for everyday math.

Hexadecimal (Base 16): Uses 0-9 and A-F. Widely used in programming and color codes.
