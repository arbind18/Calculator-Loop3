How It Works

Binary Addition: Similar to decimal addition, but carrying over when the sum is 2 (10 in binary).

Binary Subtraction: Borrowing from the next higher bit when subtracting 1 from 0.

Bitwise Operations:

  • AND: 1 if both bits are 1, else 0.
  • OR: 1 if at least one bit is 1, else 0.
  • XOR: 1 if bits are different, else 0.
