Rules

1. Letters are repeated to add value (e.g., II = 2, XXX = 30).

2. A letter cannot be repeated more than three times in a row.

3. If a smaller value is placed before a larger one, subtract it (e.g., IV = 4, CM = 900).

4. If a smaller value is placed after a larger one, add it (e.g., VI = 6, MC = 1100).
