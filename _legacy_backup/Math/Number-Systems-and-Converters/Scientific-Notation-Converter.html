Notation Types

Scientific Notation: Expressed as $a \times 10^n$, where $1 \le |a| < 10$ and $n$ is an integer.

Engineering Notation: Similar to scientific notation, but the exponent $n$ is always a multiple of 3 (e.g., $10^3, 10^6, 10^{-9}$). Useful for SI prefixes (kilo, mega, nano).

E-Notation: A way to write scientific notation where "E" or "e" replaces "$\times 10^$". Common in programming and calculators.
