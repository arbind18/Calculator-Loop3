Understanding Percentages

A percentage is a number or ratio expressed as a fraction of 100. It is often denoted using the percent sign, "%".

Common Formulas

Percentage Change: ((New Value - Old Value) / |Old Value|) × 100
Percentage Difference: (|V1 - V2| / ((V1 + V2)/2)) × 100
Reverse Percentage: Original = Final / (1 ± Percentage/100)

Quick Tips

  • Use % Change for growth/decline.
  • Use % Difference when comparing two values.
  • Reverse % finds the original price before tax/discount.
