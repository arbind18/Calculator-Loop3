# Math category (Ganit)

This folder is a **category index** for offline Math capabilities.

## What to ask (examples)

- Numbers:
  - `analyze number: 123456789012345678901234567890`
  - `add: 999999999999999999 + 888888888888888888`
  - `sqrt: 1815 20 decimals`
- Algebra:
  - `x + 1/x = 5, find x^3 + 1/x^3`
  - `a + b + c = 0 prove a^3 + b^3 + c^3 = 3abc`
- Trig proofs:
  - `prove tan((A+B)/2) = (sinA+sinB)/(cosA+cosB)`
- Geometry area/volume:
  - `tarbuj ka kshetrafal base=10 height=6`
  - `cube ka kshetrafal side=5`
  - `cuboid ka aayatan l=10 w=5 h=3`
